🐱 깃허브 용어 사전 🐱

초등학생도 쉽게 이해하는 깃허브 세상!
🎉 환영해요!
깃허브는 프로그래머들이 코드를 저장하고 함께 일하는 마법의 공간이에요!
어려운 용어들을 쉬운 말로 설명해드릴게요 😊
📁 리포지토리 (Repository)
쉬운 말로:
코드를 보관하는 특별한 폴더예요!
비유하면:
학교 사물함 같아요. 내 숙제와 교과서를 안전하게 보관하고, 친구들도 볼 수 있게 할 수 있어요.
예시:
"게임 만들기" 리포지토리에는 게임 코드와 그림 파일들이 들어있어요.
💾 커밋 (Commit)
쉬운 말로:
작업한 내용을 저장하는 것이에요!
비유하면:
게임에서 세이브하는 것과 같아요. "여기까지 했어!"라고 기록을 남기는 거예요.
예시:
"버그 수정함", "새로운 기능 추가함"이라는 메모와 함께 저장해요.
🌿 브랜치 (Branch)
쉬운 말로:
메인 코드에서 갈라져 나온 새로운 길이에요!
비유하면:
나무가지처럼 메인 줄기에서 새로운 가지가 나는 것 같아요. 실험하고 싶을 때 사용해요.
예시:
"새로운-기능" 브랜치를 만들어서 안전하게 실험할 수 있어요.
🔀 머지 (Merge)
쉬운 말로:
갈라진 브랜치를 다시 합치는 것이에요!
비유하면:
레고 블록 두 개를 합쳐서 하나로 만드는 것 같아요.
예시:
실험이 성공하면 "새로운-기능" 브랜치를 메인에 합쳐요.
📥 풀 (Pull)
쉬운 말로:
최신 버전을 내 컴퓨터로 가져오는 것이에요!
비유하면:
친구가 숙제를 업데이트했을 때, 그 최신 버전을 내가 받아오는 것 같아요.
예시:
팀원이 코드를 수정했으니 내 컴퓨터에도 업데이트해야 해요.
📤 푸시 (Push)
쉬운 말로:
내 작업을 깃허브에 업로드하는 것이에요!
비유하면:
완성한 그림을 친구들이 볼 수 있게 게시판에 붙이는 것 같아요.
예시:
내 컴퓨터에서 작업한 코드를 깃허브에 올려서 친구들이 볼 수 있게 해요.
🔍 풀 리퀘스트 (Pull Request)
쉬운 말로:
"내 작업을 확인해서 합쳐주세요!"라고 요청하는 것이에요!
비유하면:
숙제를 다 했으니 선생님께 검사받고 싶다고 손을 드는 것 같아요.
예시:
새 기능을 만들었으니 팀장님께 "확인해서 추가해주세요!"라고 요청해요.
🍴 포크 (Fork)
쉬운 말로:
다른 사람의 프로젝트를 내 계정으로 복사하는 것이에요!
비유하면:
친구의 레시피를 내 요리책에 복사해서 내 방식으로 요리하는 것 같아요.
예시:
재밌는 게임 프로젝트를 발견해서 내 계정으로 복사해서 개선해봐요.
🐛 이슈 (Issue)
쉬운 말로:
문제점이나 아이디어를 적어두는 게시판이에요!
비유하면:
학급 건의함 같아요. "이런 게 고장났어요" 또는 "이런 걸 추가해주세요"라고 적어둬요.
예시:
"게임에서 점프가 안 돼요", "새로운 캐릭터 추가해주세요" 같은 글을 써요.
🎯 재밌는 사실!
깃허브의 마스코트는 옥토캣(Octocat)이라는 고양이 문어예요! 🐙🐱
전 세계 수많은 프로그래머들이 깃허브를 사용해서 함께 멋진 프로그램들을 만들고 있어요!
